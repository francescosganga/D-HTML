# Dynamic Backend to HTML Page
 Automatically create editor for HTML page

## Getting Started

Using DB-HTML you will able to edit HTML template file from your browser.

### Prerequisites

You (will) need to have:
* Apache Server (With PHP Support)
* MySQL Database
